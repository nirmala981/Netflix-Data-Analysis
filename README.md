# Netflix Data Analysis Using Python

## Project Overview
This project performs Exploratory Data Analysis (EDA) on the Netflix Movies and TV Shows dataset to identify trends in content distribution, genres, and country-wise production.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Analysis
- Movies vs TV Shows distribution 
- Top countries producing Netflix content 
- Content release trends over the years
- Most popular genres on Netflix
- Ratings distribution

## Key Insights
- Netflix has more Movies than TV Shows.
- United States produces the highest number of Netflix titles.
- Content production increased significantly after 2015.
- Drama and Comedy are the most popular genres.

## Dataset
Netflix Movies and TV Shows Dataset
